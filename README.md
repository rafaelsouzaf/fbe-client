## Solution

- Angular 9.x
- Typescript 3.8

## Running

```
# Clone the repository
git clone https://github.com/rafaelsouzaf/fbe-client.git
cd fbe-client

# NVM
nvm use

# Install dependencies
npm install

# Start App
ng serve --open
```
